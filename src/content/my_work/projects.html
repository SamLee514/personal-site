Morsel

A CLI tool to play around with and practice Morse code. I'd been wanting to jump on the Rust hype train for a while and this turned out to be a quick and fun intro to the language. The concept was fairly straightforward: short and long key-presses correspond to dots and dashes in Morse code; Morse code lends itself well to being represented as a tree (although in hindsight a {morse_sequence : character} map would probably have been easier). Surprisingly, the most difficult part was interpretting keystrokes. As the terminal doesn't differentiate between a long key-press and a series of very quick taps, and different machine have different delays between when a key is first pressed and when repeating keys beging to be registered, there were quite a few considerations to be made.

Transfer Learning Autoencoders

A brief experiment exploring the viability of pretrained UNet-like image segmentation models as autoencoders for defect detection tasks. The goal of an autoencoder network is to learn important features from an input such that said input can be compressed and reconstructed as close to the original as possible. The reasoning behind using such networks for defect detection is that, when trained on images of "good" data, an autoencoder network will not be able to reconstruct any defects in a "bad" image (it should effectively have compressed those away). Taking a residual between inputs and outputs can thus isolate the defective area if it exists. This experiment's purpose was to see if image segmentation networks that look like autoencoders can function as such, and if so, if pretraining those networks provide any benefit.
